# Networking From Scratch: VPC, Calico, Traefik, and CloudFront on AWS

Write a technical article about building the complete networking stack. Cover the end-to-end request path (Browser → CloudFront → NLB → Traefik → K8s Service → Pod), VPC design with 3 public subnets, Calico CNI and NetworkPolicy, Traefik IngressRoute, NLB with ACM certificates, CloudFront distribution, security group matrix, and the decision to use public subnets only.
